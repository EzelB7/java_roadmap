# Ciclos en java 

## Ciclo for 
- Estructura convencional 
- Para separar los 3 bloques se usa " ; " 
``` bash 
for(inicio o desde ; fin o hasta ; con paso, salto o incremento) 
``` 
- El i++ significa 
```bash 
i=i+1;
``` 
- Ejemplo
``` bash 
for (int k = 1 ; k<=5 ; k++){
    System.out.println(k);

}
```

## Ciclo while 
- Estructura convencional 
``` bash 
while(condicion) {
    hacer 
}
```
