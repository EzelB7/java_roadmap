# Condicionales en java 

## IF 
- Estructura convencional 
- Recordar abrir y cerrar las llaves { } en un bloque if, else if y else!
``` bash 
if (condicion/es) {
    bloque de codigo
}
```
## else 
``` bash 
else {

}
``` 
## else if
- Hay que abrir y cerrar las llaves y los parentesis.
``` bash 
else if () {

}
```
