# Funciones/metodos static en java 

## Estructura convencional 
### Funcion de clase 
``` bash 
modificador_de_acceso static tipo_de_retorno nombre (tipo_de_dato parametro )
```

- El "static " hace que la funcion pertenezca a la clase y se pueda invocar sin crear un obejto 

### Modificadores de acceso 
- public = Accesible desde cualquier lado 
- private = Accesible solo dentro de la clase 
- protected = Accesible por herencia o paquete 
- default = Accesible por el mismo paquete 

### Tipos de retorno 
- int 
- String
- boolean
- double 
- objeto 
### Ejemplo 

``` bash 
public static void saludar () {
    System.out.println("esto es una funcion") ;
    }
```
