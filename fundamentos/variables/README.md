# Fundamentos java

## Nombres de archivo 
- Tu archivo debe tener el mismo nombre que tu clase para que pueda ejecutarse.
``` bash 
public class Nombre_De_Clase
``` 

## Variables 

### String
- Sirve para guardar textos 
``` bash 
- String nombre = "Ezel"
```

### int
- Sirve para guardar numeros enteros 
``` bash
int edad = 22
``` 

### double
- sirve para guardar numeros con decimales
``` bash 
double estatura = 1.70
```
### boolean
- Sirve para sentencias verdadero/falso
``` bash 
boolean estudiante = true;
```
