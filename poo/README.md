# POO en java 
- Una clase consta de atributos y metodos 
- Los atributos son como las caracteristicas o especificaciones que va llevar tu clase 
- Los metodos son las funciones o habilidades de tu clase 
- Cuentan con una funcion "constructor" para inicializar tus atributos. y no tiene tipo de retorno.
## Estructura general de los atributos 
``` bash 
public class nombre_clase{
atributos 
String, int, bool, etc. nombre_de_atributo
}
```
